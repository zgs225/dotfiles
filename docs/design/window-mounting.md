# 画心框绫 —— i3 窗口装裱

> 状态：已实施（2026-07-22）
> 上游：`song-liquid-glass.md` §1（装裱隐喻）§2.1（令牌）§6（i3 边框色）

## 1. 问题

`client.focused` 五字段全天青，每个聚焦窗口顶部是一条 100% 宽的实心天青横幅——全局面积最大的高饱和色块。天青的系统语义是「界引/框绫」（发丝级用色），横幅违反 §1.2 克制纪律；且装裱隐喻在画心处断裂：立轴（bar）、多宝格（弹层）皆已入裱，唯独窗口本身还是装饰器默认。

## 2. 裁定（2026-07-22，用户）

- focused 标题栏**降格为框绫**：去横幅，天青只剩 2px 框。
- **picom opacity 不动**（active 0.85 / inactive 0.70 保留，用户喜欢现状手感）——纵深不由 opacity 承担。
- 折中案（横幅沉 `#7fa8a0`）否决。

## 3. 方案

隐喻映射：窗口 = 画心，2px 边框 = 框绫，标题栏 = 题款。纵深走**题款三档色阶**：

| 状态 | border / indicator / child_border | titlebar bg | 题款文字 |
|---|---|---|---|
| focused（画心） | 天青（不变） | 天青 → **玄 `bg_base`** | 玄底黑字 → **月白 `fg_primary`** |
| focused_inactive | 墨（不变） | 墨（不变） | 月白 → **蟹壳青 `fg_secondary`** |
| unfocused（淡题款） | 墨（不变） | 玄（不变） | 蟹壳青 → **蟹壳青暗阶 `#85857d`**（衍生固定值，不入令牌） |

urgent 赭石、placeholder、picom、opacity-rule 全部不动。

聚焦信号 = 天青发丝框 + 最亮题款。题款色阶（月白 > 蟹壳青 > 暗阶）为第二根纵深轴。聚焦标题栏与 bar 落款区同气质（玄底 + 月白文楷）。

## 4. 验证

1. `chezmoi apply` → `i3-msg reload`（不重启 eww）。
2. 截图目视：全屏无天青横幅；三档题款可辨。
3. 像素采样确认朱砂仍唯一（I3 惯例）。
4. 回写 `song-liquid-glass.md` §6「i3 边框色」行，补题款三档纪律。

## 5. 非目标

- floating 窗口差异化（册页/黛底题款）——登记观察项，后续想更戏剧时再做。
- i3 边框 alpha（X11 实色足够，不引入不确定性）。
- cellPad / 微间距等既有登记项，不在本方案。
