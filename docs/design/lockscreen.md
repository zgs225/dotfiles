# 锁屏设计 · 立轴

> 本文件仅描述锁屏的视觉设计语言。技术实现见 `dot_config/i3/scripts/`（`lock-render.py` / `lock.sh`）及 `dot_config/systemd/user/lockscreen-refresh.*`。
> 色彩令牌、字体方案、设计纪律遵循 `song-liquid-glass.md`。

---

## 1. 立意

锁屏是一幅立轴。

用户合盖或离席，屏幕不是黑屏，是展开一卷绢本：右上题诗、钤印，左下纪时，中央留一道解锁的环。

取法宋人册页的不对称构图——大面积留白（模糊壁纸）承托少量文字，信息密度极低，呼吸感极强。

---

## 2. 构图

画幅即屏幕（2560×1600 基准），分三个视觉重心：

```
┌─────────────────────────────────────────────┐
│                                             │
│                          竖排诗句（右上）     │
│                          │                  │
│                          │                  │
│                          干支印              │
│                                             │
│                                             │
│                                             │
│              解锁环（居中偏下）               │
│                                             │
│                                             │
│   时辰大字（左下）                            │
│   公历小字                                   │
│                                             │
└─────────────────────────────────────────────┘
```

- **右上**：竖排诗句，自上而下书写，占画面右缘约 1/5 宽度
- **诗句下方**：一方干支印，朱文（字凸），赭石色
- **左下**：时辰二字横排，大字；其下公历日期，小字
- **居中偏下**：解锁环，画面几何中心略偏下（约 55% 高度处）

三组元素互不重叠，各据一角，中间大片模糊壁纸即"绢面"。

---

## 3. 元素

### 3.1 竖排诗句

- 每次锁屏从十二句语料中随机取一句
- 竖排，自右向左（传统书写方向），单列
- 内容为五言或七言古典诗句，取意境清冷、与"夜/静/归/锁"相关者
- 字色：自适应（max-contrast 选色，逐字测模糊底图局部亮度）——亮底自动切题字墨 `ink_text #202024`，暗底保留月白 `fg_primary #e8e6df` 的"隔纱观物"低对比。逐字切换因模糊底图低频而稀少，落在画里自然明暗交界，无跳色、无硬补丁
- 字距略松，取宋刻本疏朗感

### 3.2 干支印

- 位于诗句末字下方，间距约两字高
- 形制：竖式方印（宽随单字、高随字数自适应），圆角 3px（全局圆角纪律）
- 印文：当日干支两字**竖排**（自上而下），白文（字凹，底色为印色）
- 字体：`Chong Xi Small Seal`（崇羲篆體）22px——方印取篆书，呼应印章语义
- 印色：赭石 `#a0522d`（非朱砂——朱砂唯一化纪律，见 `song-liquid-glass.md` §1.2）
- 尺寸：宽 ≈ 单字宽 + 2×pad，高 ≈ 两字高 + 字隙 + 2×pad（pad=8）；不再固定方形

### 3.3 时辰

- 左下，横排，二字（如"戌时"）
- 字体：Source Han Serif CN Bold，72px
- 字色：自适应——亮底题字墨 / 暗底月白（实现见 §3.1）
- letter-spacing 2px，取宋体筋骨
- 定格于锁屏瞬间，不随时间跳动

### 3.4 公历

- 时辰正下方，一行小字
- 格式：「二〇二六年七月二十一日」或「2026 · 07 · 21」
- 字体：LXGW WenKai Regular，16px
- 字色：自适应——亮底次阶墨 `ink_text_dim #3a3a34` / 暗底蟹壳青 `fg_secondary #9a9a92`（次要文字层级）
- 与时辰大字形成主/次对比

### 3.5 解锁环

- 居中偏下，半径 80px，环宽 4px
- 常态：天青 `#9ec8c0`，alpha ≈ 0.27（极淡，近乎隐入绢面）
- 验证中：天青，alpha 提升
- 退格：藤黄 `#d8a23c`
- 错误：赭石 `#a0522d`
- 无填充，纯线环；内部无文字（时辰已在左下）

## 4. 色彩

| 元素 | 色（自适应） | 说明 |
|---|---|---|
| 诗句 | 月白 / 题字墨 | 暗底月白守"隔纱观物"；亮底题字墨保可读（max-contrast 选色，逐字切换） |
| 干支印底 | 赭石 `#a0522d` | 非朱砂，纪律合规 |
| 干支印文 | 月白 `#e8e6df` | 白文，自带实底不依赖壁纸 |
| 时辰大字 | 月白 / 题字墨 | 同诗句，自适应 |
| 公历小字 | 蟹壳青 / 次阶墨 | 次信息；亮底切次阶墨 |
| 题字墨 token | `ink_text` `#202024` | `fg_primary` 的暗字双胞胎 |
| 次阶墨 token | `ink_text_dim` `#3a3a34` | `fg_secondary` 的暗字双胞胎 |
| 解锁环 | 天青 `#9ec8c0` @ 27% | 极淡，不抢视觉 |
| 壁纸 | 模糊原图 | 绢面底色 |

全画面无朱砂、无藤黄（除非解锁交互态）、无高饱和色。静。

---

## 5. 字体

| 元素 | 字体 | 字号 | 字重 |
|---|---|---|---|
| 竖排诗句 | Source Han Serif CN | 36px | Regular |
| 干支印文 | LXGW WenKai | 22px | Bold |
| 时辰 | Source Han Serif CN | 72px | Bold |
| 公历 | LXGW WenKai | 16px | Regular |

诗句与时辰用宋体（刻本筋骨），印文与公历用文楷（手写温度）。两族字体交替，避免单调。

---

## 6. 留白与比例

- 诗句区占画面面积 < 8%
- 时辰区占画面面积 < 5%
- 解锁环占画面面积 < 3%
- 其余 > 84% 为模糊壁纸（绢面）
- 三组元素之间最小间距 > 200px
- 诗句距右缘 ≈ 屏宽 × 0.08；距顶缘 ≈ 屏高 × 0.10
- 时辰距左缘 ≈ 屏宽 × 0.06；距底缘 ≈ 屏高 × 0.12

原则：元素是"题"在绢上的，不是"铺"满的。

---

## 7. 动态与静态

| 元素 | 变化时机 | 说明 |
|---|---|---|
| 诗句 | 每次锁屏随机 | 十二句轮换 |
| 干支 | 每日变化 | 定格于锁屏瞬间 |
| 时辰 | 每两小时变化 | 定格于锁屏瞬间 |
| 公历 | 每日变化 | 定格于锁屏瞬间 |
| 壁纸 | 用户更换 | 模糊处理 |
| 解锁环 | 交互态 | 随输入反馈变色 |

锁屏后画面完全静止，无动画、无时钟跳动。静如裱好的立轴。

---

## 8. 设计纪律

1. **朱砂唯一化**：锁屏全画面不得出现 `#c8452c`。印章用赭石。
2. **无对称**：三组元素分据右上、左下、中偏下，拒绝居中对称排版。
3. **无装饰纹样**：不加云纹、回纹、边框。留白即装饰。
4. **无动画**：锁屏是静态画面，不做淡入、不做呼吸灯。
5. **隔纱 vs 可读**：暗底文字守"隔纱观物"低对比（淡月白、不刺目）；浅色底文字破例切墨字（题字墨 / 次阶墨）保证可读。逐字 max-contrast 选色——免阈值调参，模糊底图天然低频使切换稀少且落在画里自然明暗交界，不露补丁。
6. **圆角纪律**：唯一有圆角的元素是干支印（3px），其余均为直角或无边界。
